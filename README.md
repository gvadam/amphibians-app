# Amphibians

- RecyclerView
- Viewbinding
- Viewmodel
- Retrofit
- Moshi
- Navigation Component

Data on amphibians are retrieved from an API and used to populate the home screen with a list. Each amphibian card is clickable and takes the user to a detail screen.
